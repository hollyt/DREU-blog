Week 5 - 06.23.14 - 06.27.14

During the beginning of this week I worked with the openSMILE toolkit since I was still waiting to receive the correct data set. We plan to use openSMILE for real-time audio feature extraction in our virtual human, so I started working on implementing that.

openSMILE needs the PortAudio library for real time extraction, but the PortAudio that comes with the newest version of the Windows configuration is out of date and doesn’t compile. I ended up downloading PortAudio directly and then tweaking some of the openSMILE source code so it compiles with the newest version of PortAudio. It worked!

I eventually received the correct data for the Friendship Classifier, but we also figured out that the model of Friendship Prediction from one of the lab members will not work for our project, so I’m no longer basing my classifier on that model.

That means that all I have now is my classifier program and the raw data, which has thousands of data points and thousands of features. I am now in the process of finding the cost value for my classifier that yields the highest accuracy.
