Week 2 - 06.02.14-06.06.14

My Friendship Classifier will use a Support Vector Machine (SVM) to classify and predict friendship. I’m new to Machine Learning, and I’ve never had any experience with implementing SVMs, so I spent most of this week educating myself on the subject.

I watched some lectures from Stanford's Machine Learning course on Coursera, read through some slides from the Machine Learning course at CMU, and studied the documentation for libsvm. I’m happy to say that I now know (just the tip of the iceberg!) how to use Support Vector Machines to solve classification problems, and I’m eager to learn more about Machine Learning in general.

This week’s reading list focused on work that has been done outside of the lab on creating rapport between humans and virtual humans.
