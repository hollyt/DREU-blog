Week 1 - 05.26.14 - 05.30.14

My first week at the ArticuLab was mostly filled with getting oriented with the lab and my project. I’ll be working on the Rapport Project - a project that aims to create a virtual human peer tutor that can build, maintain, and break rapport in order to help facilitate learning geometry. Specifically, I will be working on the Friendship Classifier, which is a machine learning classifier that uses multimodal input to predict whether or not two conversationalists are friends or strangers. Eventually this classifier will be implemented as a module in the Virtual Human Toolkit and will be used in the Rapport Project’s virtual peer to decide which rapport strategies the virtual human should use depending on whether or not the human and virtual human are friends.

I read some papers from members of the lab about Friendship Prediction, a Dyadic Computation Model for Rapport Management, and a Computational Architecture of Dyadic Rapport Management so I could get a feel for what has already been done on the Rapport Project, as well as the ultimate goals of the project.

I also familiarized myself with some of the tools that I will be working with in order to implement my Friendship Classifier: VHtoolkit, Matlab, openSMILE, libsvm.
