About My Mentor

Justine Cassell (Ph.D., University of Chicago) has joined the Human-Computer Interaction Institute at Carnegie Mellon University as Director. Cassell comes to CMU from Northwestern, where she was the founding director of the Technology and Social Behavior joint PhD in Communication and Computer Science, and of the Center for Technology and Social Behavior. Before Northwestern, Cassell was a tenured faculty member at the MIT Media Lab, where she headed the Gesture and Narrative Language research group. Cassell's research focuses on understanding natural forms of communication, and then creating technological tools for those forms of communication and linguistic expression to flourish in the digital world. In particular, she is credited with developing the Embodied Conversational Agent, a virtual human capable of interacting with humans using both language and nonverbal behavior. More recently Cassell has investigated the role that the ECA can play in children's lives, as a part of a Story Listening System, an interactive support for learning language and literacy skills.

More information is available on Justine's website.



About My Project

I'm working on the Connection Machines Project, which is typically refered to as the Rapport Project within the lab. Based on research that suggests rapport between two peers benefits learning in a peer tutoring situation, the Rapport Project aims to create a computational model of rapport management, and ultimately create a virtual agent that is about to create, maintain, and build rapport with a human user in real-time.

My task within the Rapport Project is to implement the Friendship Classifier, which is a module that will be used to build the virtual agent peer tutor. It is a binary module that decides whether or not the virtual agent and human peer are friends when given multimodal input such as the peer's smile, gaze, and speech. The system will be trained using data from the recorded peer tutoring sessions and will learn using a support vector machine. My ultimate goal is to create a module that will interact with other modules that execute Intention Understanding, Rapport Generation, Behavior Planning, etc. to create a virtual agent that can create, maintain, and break rapport with a user in order to facilitate a better learning environment.
